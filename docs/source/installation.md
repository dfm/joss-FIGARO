# Installation

You can install FIGARO either via pip (stable release, preferred)

```
pip install figaro
```

or from its source code (potentially unstable)

```
git clone git@github.com:sterinaldi/FIGARO.git
cd FIGARO
pip install .
```
## Dependencies

FIGARO has the following dependencies:

* numpy > 1.22
* scipy
* matplotlib
* dill
* Corner
* Numba
* h5py
* tqdm
* lalsuite
* ray
